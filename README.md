# Create-an-Android-app-3

**어플의 기능 및 구현**
1. 핸드폰에서 사용할 수 있는 센서목록을 다 출력해준다. 그 중에 몇가지의 센서를 테스트 해본다
2. gps의 데이터 값을 가져와 확인해본다.
3. 카메라에 접근하여 사진을 찍은 후 결과창을 보여주고 그것을 저장해 리스트로 출력해준다.
